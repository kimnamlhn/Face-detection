# Face-detection

Build Real Time Face Detection With JavaScript

## Usage

Step 1: Clone this repo 

```
$ git clone https://github.com/kimnamlhn/face-detection
```

Step 2: Open it in VS code and run with live server

Step 3: See it in browser

## Technologies

- HTML, CSS, JS
- Face-api
